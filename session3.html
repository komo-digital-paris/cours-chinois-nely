🧭 Adresses chinoises — du grand au petit : pays → ville → rue → numéro (中国 · 上海市 · 南京路 · 123号). C'est l'inverse du français ; les dates suivent la même logique (année → mois → jour).

Jeu de rôle Le touriste perdu & le GPS humain (2 joueurs)

🎭 Le « non » chinois : refuser directement (不) est brutal. Si on vous répond « pas très pratique » ou « je vais réfléchir », c'est souvent un non poli. En affaires, ne poussez jamais après un refus déguisé.

Section 02 · Exercices

Exercices — Questions pratiques

À trous Le bon mot

什么 / 怎么说 / 听不懂 / 可以 / 有没有

1. 请问，这是___？

2. « Merci » ___？

3. 我___，请再说一次。

4. ___坐这里吗？

5. ___不辣的？

Vrai / Faux Code culturel

1. Dire 不 directement est poli.  V / F

2. 我考虑一下 = souvent « non ».  V / F

3. 有没有 demande si ça existe.  V / F

4. Pas de 好 clair = il est d'accord.  V / F

Jeu de rôle Le restaurant mystère & le traducteur improvisé (2 joueurs)

A — Client : tu ne comprends rien au menu. Enchaîne : 这是什么？辣不辣？有没有不辣的？我要这个。

B — Serveur : explique les plats (invente !), parle vite puis lentement quand A dit 请再说一次. Variante : montrez un objet et demandez 这是什么？___怎么说？

✅ Corrigé

À trous : 1. 什么 · 2. 怎么说 · 3. 听不懂 · 4. 可以 · 5. 有没有

Vrai / Faux : 1. F (c'est brutal) · 2. V · 3. V · 4. F (refus poli)

Section 03 · Exercices

Exercices — Paiement

Vrai / Faux Le paiement

1. La plupart des commerces acceptent le cash.  V / F

2. 扫码 = « scanner le QR ».  V / F

3. Un hongbao de 444 元 est un bon cadeau.  V / F

4. 块 et 元 = le yuan.  V / F

Remets en ordre Les étapes

A. 收到了，谢谢！

B. 买单！

C. 微信还是支付宝？

D. Tu scannes le QR code

E. 微信支付。

Jeu de rôle Simuler un paiement (2 joueurs)

A — Caissier : annonce un prix (一共六十块), demande 微信还是支付宝？ puis 请扫码。

B — Client : répond, mime le scan, dit 好了！ Le caissier confirme 收到了，谢谢！ Recommencez 3× avec des montants différents.

✅ Corrigé

Vrai / Faux : 1. F · 2. V · 3. F (4 = malchance, évitez) · 4. V

Ordre : B → C → E → D → A
